Teacher Instructions

In order for your students to use LanguageBot, they will need an API key, which you will have to give them. To get an API key, follow these instructions:

Get API Key

1. Go to Google AI Studio

2. Sign in with your PERSONAL google account (If you use a school account, you may not be able to access Google AI Studio)

3. On the lower-left corner, click "Get API Key"

4. On the upper-right corner, click "Create API Key"

5. Name your key and create a project (if needed)

6. After you've created your API key, copy it and email it to your students

NOTE: DO NOT SHARE YOUR API KEY WITH ANYONE ELSE BESIDES YOUR STUDENTS!
